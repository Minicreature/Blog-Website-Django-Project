# Blog Website - v1 (Initial Stage)

This is v1.0.0 - Initial Django project scaffold and Post model

What's included:
- Django project created
- blog app added
- Post model (title, content, author, date_posted)
- initial migrations and sqlite DB

How to run (local):
1. python -m venv venv
2. source venv/Scripts/activate
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py runserver
